Computer Look-Inside

Picture + Words

  • CPU (Central Processing Unit)
    Fancy Word for Fan. The CPU keeps the computer cool so that is doesn't overheat.
  • Power Supply Unity
    This powers everything inside the computer. Near the Power Supply Unit, you find a spot to plug it into an outlet.
  • Motherboard
    The Motherboard stores all the computer's information. All the apps, files, pictures, games, and documents are stored here. (All the Software)
  • GPU (Graphic Processing Unit)
    The GPU connects to the monitors and makes all the software visible to us. When you hear someone say, "That game has terrible graphics," it may be because there's a problem with the GPU. It refers to what we're looking at.
  • RAM (Random Access Memory)
    The RAM communicates between the Motherboard and the GPU. It converts that information (software) into something that we can see.
  • Hard Drive
    The Hard Drive holds additional information, like a USB Drive. For intances, if you have an game on a CD that you want to play on your computer, you would put it in the hard drive.
