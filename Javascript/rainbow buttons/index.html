Rainbow Button

The Setup

  1. Create your HTML and JavaScript pages.
  2. Set up your HTML page
    1. Link you JavaScript page
    2. Insert jquery cdn
    3. Put a button in the center of the screen
    4. Make sure your button has either a class or id.
  3. Start coding in JavaScript

The JavaScript

  1. start with your wrapper function.
    1. The purpose of our wrapper function is to keep all of our code together
                      
            (function() {

            })();
  2. Now add the document ready statement
    1. This tells the computer to wait until our document is ready before it starts reading the JavaScript page.
                      
            (function() {

              $(document).ready(init);

            })();
  3. The next step is tell the computer what to do when the button is clicked
    4.                     
            (function() {

              $(document).ready(init);

              function init() {
                $("#button").click(red);
              }

            })();
    1. This line of code says, "when the id of button is clicked, run the function red."
  4. Next, in the function red, we actually want to change the color of the text to red.
    1. This line says that the id, button, will change its css color to red.
    2. Then, the we want to say, "If the id—button is clicked again, then the run the function orange"
                    
          (function() {

            $(document).ready(init);

            function init() {
              $("#button").click(red);
            }

            function red() {
              $('#button').css('background-color', 'red');
              $('#button').click(orange);
            }

          })();
  5. After the button is clicked again the color will change to orange.
