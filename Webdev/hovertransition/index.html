Hover Transitions

The Setup

  1. Create your HTML and CSS pages.
  2. Set up your HTML page
    1. Link you stylesheet
    2. Put a button in the center of the screen
    3. Make sure your button has either a class or id.
  3. Start coding in CSS

CSS

  1. Give your button a list of styling properties.
    2.             
              #button {
                color: black;
                background-color: orange;
                padding-top: 50px;
                padding-bottom: 50px;
                border: 10px solid black;
              }
  2. Now all you have to do is change what you what using the hover pseudo class.
    3.                 
              #button {
                color: black;
                background-color: orange;
                padding-top: 50px;
                padding-bottom: 50px;
                border: 10px solid black;
              }

                
              
                    
              #button:hover {
                color: :orange;
                background-color: :black;
                padding-top: :75px;
                padding-bottom: :75px;
                border: :20px dotted white;
              }
